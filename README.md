## 🔥 Key Features

Stock software limits frequencies to conserve battery and keep physical surfaces cool. Apex Kernel Hub opens the floodgates:

* **Pure Performance Focus:** This release focuses solely on delivering raw power. **Undervolting is NOT supported**—frequencies run at stable high-voltage thresholds.

### 2. Complete Thermal Limit Removal
BBK software is notoriously aggressive with thermal throttling, often dropping display refresh rates or reducing game performance the instant the phone warms up.
* **Thermal Daemon Bypass:** Deactivates system-level thermal managers tracking device temperatures.
* **Unthrottled Power:** Keeps your device running at full speed regardless of temperature thresholds.
* *Note: Your device will get noticeably hot under heavy load. It is highly recommended to use an active external phone cooler to prevent hardware degradation.*

### 3. Hardware Bypass Charging
A critical feature for long-duration mobile gaming and power users.
* **Direct Rail Power:** Once enabled, the application hooks into the native BBK charging logic to route external cable power directly to the motherboard.
* **Thermal Control:** By bypassing the battery, the phone cuts down the massive heat normally generated by charging, preserving long-term battery health and keeping the CPU/GPU cool.

### 4. Edge-to-Edge Fullscreen AOD
Break free from tiny, static, pre-configured Always-On Display regions.
* **Full Canvas Utilization:** Forces the stock AOD subsystem to exploit the entire screen real estate.
* **Deep UI Overrides:** Prevents system daemons from aggressively fading out or minimizing full-screen ambient configurations.

### 5. Granular 4-Level Animation Engine
Ditch the restrictive Developer Options scaling. This APK injects custom interpolation factors directly into the window management framework across four distinct levels:

* 🚀 **Level 1 (Hyper-Fast):** Near-zero transition delays. Everything snaps instantly for an ultra-responsive, instantaneous feel.
* ⚡ **Level 2 (Snappy):** Accelerated UI speeds that eliminate lag while retaining minimal visual framing for clean navigation.
